About Us!

This website was designed, written and developed by the kids of Blackheath Coding Club. We meet every Sunday at 5:30 and learn how to code. We have two teachers to help us, Rob and Lily, who are very enthusiastic and fun to learn with! At first coding wasn't as serious because it was just a fun activity to do on a Sunday evening, but it started to get serious when we decided to enter Young Rewired State.

What we've learned:

CSS, HTML and a little of JavaScript. HTML stands for Hyper Text Markup Language. Whenever making a website HTML is the first and last tag on the page, but on the last tag you put a forward slash, like this: . CSS is our team's favourite part. It makes a website attractive and easy to read.CSS is the colours and the font of a website.It was a major part in making this site. We have learnt a lot about coding over the last few months. On our first lesson we learnt about; the user, the web browser, the Internet and the back end as you can see in the drawing below.

What we've done:

First we made a web page, which consisted of pictures and writing (copy), then we started to develop links, background colours and the position of these things. During the Christmas period some of us created a christmassy web page including videos, copy, music and colour. To make these web pages, we used notepad and then later we installed Sublime Text 2.
